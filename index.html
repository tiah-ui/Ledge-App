App & Updates
Manage the Ledge app file and install updates
📦 Current Version
📄
Ledge App File
Not located yet
v7.0
Ledge is a single HTML file — Ledge.html — saved on your device.
To update, you download the new version, then use the button below to replace your current file automatically.
1 Locate Your App File
Tell Ledge where your Ledge HTML file is saved on your computer (e.g. your Downloads or Desktop folder).
You only need to do this once — Ledge remembers it.
2 Apply an Update
Once you've downloaded a new version of Ledge:
1. Click "Upload New HTML File" below.
2. Select the newly downloaded Ledge HTML file.
3. Ledge will overwrite the old file and reload with the new version.

⚠️ Step 1 must be completed first. Your data is never affected by updates.
📲 Install as App
Add Ledge to your home screen or taskbar so it opens like a native app — no browser tab needed.
